##### Project Name : Raspberry Pi 차량 출입 관리 시스템

##### Server>>Ubuntu 16.04

##### Client>>Raspberry pi 3 model rev 1.2(Raspbian Buster),opencv 4.1.2,Raspberry pi Camera v2 Module

##### 1 기능 : 8자리 차량 번호판 추출

##### 2 기능 : 차량 번호판 추출 차량 번호 인식 

##### 3 기능 : 등록된 차량번호 확인,인식된 번호와 저장된 데이터 비교 후 처리
